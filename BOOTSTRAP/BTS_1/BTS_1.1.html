A cosa serve bootstrap?

Bootstrap è una libreria di componenti front end e un toolkit open source.

Viene usato per creare pagine web reattive e mobile-first con HTML, CSS e JavaScript.

Cos’è un CDN? Che vantaggi comporta rispetto a scaricare bootstrap sul tuo sito?

Una CDN (Content Delivery Network) è una rete di server che distribuisce il contenuto da un server "di origine" in tutto il mondo memorizzando nella cache il contenuto vicino al punto in cui ogni utente finale accede a Internet tramite un dispositivo abilitato al Web

la maggior parte dei CDN si assicurerà che una volta che un utente richiede un file da esso, verrà servito dal server più vicino, il che porta anche a tempi di caricamento più rapidi.

A cosa serve questo attributo del tag meta: content="width=device-width, initial-scale=1?

Il width=device-width imposta la larghezza della pagina per seguire la larghezza dello schermo del dispositivo (che varierà a seconda del dispositivo).

initial-scale=1 imposta il livello di zoom iniziale quando la pagina viene caricata per la prima volta dal browser.

Che differenza c’è tra Container e Container Fluid (prova il container fluid creando una pagina chiamata BTS_1.1.1.html)?

La classe .container fornisce un contenitore reattivo a larghezza fissa

La classe .container-fluid fornisce un contenitore a larghezza intera, che copre l'intera larghezza della finestra
